# React Practice Projects

This repository contains a collection of React projects created for practice and learning purposes. Each project is in its own directory.

| Project Name | Description |

| [`forum-codex`](./forum-codex/) | A basic React forum application. |
| [`forum-cursor`](./forum-cursor/) | A basic React forum application. |
| [`react-practice`](./react-practice/) | A project for general React practice and experiments. |

| [`tic-tac-toe-codex`](./tic-tac-toe-codex/) | A simple Tic-Tac-Toe game built with React. |
| [`tic-tac-toe-cursor`](./tic-tac-toe-cursor/) | A feature-rich Tic-Tac-Toe game with move history, win highlighting, and more. |
